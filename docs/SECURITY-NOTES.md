# Security Notes

このツールは、Outlook予定表をHTML/CSVとして別フォルダに出力します。便利ですが、予定表情報は機密性が高い可能性があります。

## 推奨設定

- 出力先フォルダは自分専用、または必要最小限の権限に限定する
- 非公開予定は件名も場所も出さず、`非公開予定` と表示する
- まずは `DaysAhead` を 7〜14日程度にする
- ログには予定内容を書かない
- 本文プレビューは必要な場合だけ有効化する

## 詳細表示で出力され得る情報

設定によって、以下の情報が `schedule.html` と `schedule.csv` に保存されます。

- 主催者
- 必須参加者
- 任意参加者
- 場所
- 分類
- リマインダー
- 本文プレビュー

特に参加者名、メールアドレス、会議本文、Teams URL、案件名、取引先名は機密情報になり得ます。

## 既定の安全寄り設定

- `MaskPrivateItems: true`
- `IncludeBodyPreview: false`

本文は機密情報を含みやすいため、既定では本文プレビューを出しません。

## 避けること

- 個人クラウドへの同期
- 広い共有フォルダへの出力
- DLPポリシーや実行ポリシーの迂回
- 承認されていない自動転送
- 本文プレビューを有効にしたHTML/CSVを広い権限のフォルダに置くこと

## 情シス相談時の説明例

目的:

Outlookを利用できない社内端末で、自分の予定を確認したい。

処理:

Outlookが利用できる端末上で、自分の予定表を30分ごとに読み取り、HTML/CSVとして、自分だけが読める社内フォルダに出力したい。

出力項目:

開始時刻、終了時刻、件名、場所、主催者、参加者、空き時間状態。本文プレビューは必要時のみ有効化。

安全対策:

非公開予定は詳細をマスクし、出力先フォルダの権限を限定する。
