# Security Notes

このツールは、Outlook予定表をHTML/CSVとして別フォルダに出力します。便利ですが、予定表情報は機密性が高い可能性があります。

## 推奨設定

- 出力先フォルダは自分専用、または必要最小限の権限に限定する
- 本文、参加者、Teams URL、添付は出力しない
- 非公開予定は件名も場所も出さず、`非公開予定` と表示する
- まずは `DaysAhead` を 7〜14日程度にする
- ログには予定内容を書かない

## 避けること

- 個人クラウドへの同期
- 広い共有フォルダへの出力
- Teams URL、会議本文、参加者一覧の出力
- DLPポリシーや実行ポリシーの迂回
- 承認されていない自動転送
